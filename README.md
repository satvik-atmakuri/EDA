2. **Clustering & Anomaly Detection**:
   - Apply clustering techniques (e.g., K-Means).
   - Detect and handle anomalies in datasets.

3. **Model Building**:
   - Use AutoVIML for ML models.
   - Build, evaluate, and compare models, including ensemble methods.
